菊のマーブル

菊のマーブルとは

菊のマーブルはWplace上に指定した画像のガイドを表示するツールです

菊のマーブルはBlueMarbleというツールを日本語化、機能の追加、UIの変更などしたユーザースクリプトです

Tampermonkeyという拡張機能を使用して動作します。

Tampermonkeyの導入方法

PCの場合

h3>
  1. 使用しているブラウザのウェブストアからTampermonkeyをインストールする
    • Chrome / Edge : Chromeウェブストア
    • Edge : Edgeアドオン
    • Firefox : Firefoxアドオン
  2. ブラウザ右上の拡張機能一覧からTampermonkeyの管理ページを開く
    • Chrome / Edge の場合: `開発者モード`や`ユーザスクリプトを許可する`などを有効化する
    • Firefoxの場合は不要

スマホの場合

  1. Edgeをインストールする
    • おそらく拡張機能が使えるブラウザがEdgeだけなのでEdgeをインストールしてください
  2. Edge画面右下の三本線から拡張機能を開いてTampermonkeyをインストールする
    • 最初にこの画面を開くと拡張機能の一覧が出るのでその中からTampermonkeyを探してください
    • 見つけたら開いて購入ボタンを押してください(購入と書いてありますが実際は無料です)

使用方法

  1. ドラッグエリア

    この範囲をクリックしながら動かすとウィンドウ全体をドラッグできます

  2. 最小化ボタン

    この範囲をクリックすると最小化と展開を切り替えられます

  3. 情報表示

    自分のユーザー名、使用できるドロップ、何ピクセル塗れば次のレベルへ行けるか、などの情報を表示します

  4. ピンの位置を座標に

    現在ピンを指している場所を画像に使用する座標にします

  5. 座標を指定

    座標を入力して指定できます

  6. 画像読み込みモード切替ボタン

    PCにあるローカルファイルを指定 / URLで画像を指定を切り替えられます

  7. 画像ファイルを指定ボタン

    使用する画像をローカルファイルから選べます

  8. 画像URLを入力エリア

    ここに画像のリンク(ディスコードでアップロードされた画像のリンク等)を書いて使用することができます

  9. 作成ボタン

    指定した座標と画像でガイドを作成します (この処理には少し時間がかかります)作成後、ガイドの表示は自動で有効になります

  10. 有効無効切り替えスイッチ

    スイッチを切り替えることで、ガイドの表示・非表示を切り替えられます（この処理には少し時間がかかります）。

  11. ログ

    ログが表示されます。右下をドラッグすればサイズを変えられます(使用することはほぼ無い)

  12. カラーコンバーターへ

    Wplaceカラーコンバーターという菊のマーブル等に使うガイド用の画像を生成できるツールへ移動します
