菊のマーブル

菊のマーブルとは

菊のマーブルはWplace上に指定した画像のガイドを表示するツールです

菊のマーブルはBlueMarbleというツールを日本語化、機能の追加、UIの変更などしたユーザースクリプトです

Tampermonkeyという拡張機能を使用して動かします

Tampermonkeyの導入方法

  1. 使用しているブラウザのウェブストアからTampermonkeyをインストールする
    • Chrome / Edge : Chromeウェブストア
    • Edge : Edgeアドオン
    • Firefox : Firefoxアドオン
  2. ブラウザ右上の拡張機能一覧からTampermonkeyの管理ページを開く
    • Chrome / Edge の場合: 「開発者モード」を有効化する
    • Firefoxの場合は不要

菊のマーブルの導入方法

現在菊のマーブル公式リポジトリがバグが発生しているので僕がフォークし修正してプルリクエストを送った段階の物を使用してください！

  1. スクリプトのURLへ行く

    現時点(2025/8/14)で正常に動くリンク

    https://github.com/gori-GORILLA-gori/ChrysanthemumMarble/raw/refs/heads/main/dist/ChrysanthemumMarble.user.js

  2. インストールする

    自動でTampermonkeyでのインストール画面になるのでインストールボタンを押しインストールする

  3. もしWplaceを開いているならリロードする

    リロードしないとインストールが適用されません

使用方法

  1. ドラッグエリア

    この範囲をクリックしながら動かすとウィンドウ全体をドラッグできます

  2. 最小化ボタン

    この範囲をクリックすると最小化と展開を切り替えられます

  3. 情報表示

    自分のユーザー名、使用できるドロップ、何ピクセル塗れば次のレベルへ行けるか、などの情報を表示します

  4. ピンの位置を座標に

    現在ピンを指している場所を画像に使用する座標にします

  5. 座標を指定

    座標を入力して指定できます

  6. 画像読み込みモード切替ボタン

    PCにあるローカルファイルを指定 / URLで画像を指定を切り替えられます

  7. 画像ファイルを指定ボタン

    使用する画像をローカルファイルから選べます

  8. 画像URLを入力エリア

    ここに画像のリンク(ディスコードでアップロードされた画像のリンク等)を書いて使用することができます

  9. 作成ボタン

    指定した座標、画像でガイドを作成します(この処理にはラグが発生します)(10が自動でONになります)

  10. 有効無効切り替えスイッチ

    スイッチを切り替えることで表示を出したり消したりできます(この処理にはラグが発生します)

  11. ログ

    ログが表示されます。右下をドラッグすればサイズを変えられます(使用することはほぼ無い)

  12. カラーコンバーターへ

    Wplaceカラーコンバーターという菊のマーブル等に使うガイド用の画像を生成できるツールへ移動します
